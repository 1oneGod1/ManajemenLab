Tentang Web Report

Dibangun untuk sekolah modern

Web Report adalah proyek manajemen laboratorium berbasis web yang berjalan penuh di browser. Menggunakan Firebase Realtime Database untuk sinkronisasi data dan Cloudflare R2 untuk penyimpanan foto, sistem ini ringan, cepat, dan gratis dijalankan.
