1. Deskripsi Aplikasi

Lab Report adalah Sistem Manajemen Laboratorium Terpadu berbasis web yang digunakan untuk mengelola fasilitas secara terpusat di empat jenis laboratorium: Komputer, Fisika, Kimia, dan Biologi. Sistem ini menggabungkan Firebase Realtime Database, Firebase Authentication, dan Cloudflare R2 sebagai infrastruktur cloud tanpa server (serverless/BaaS).

Aplikasi dibangun menggunakan Vanilla HTML, CSS, dan JavaScript tanpa framework atau build tool, sehingga dapat berjalan langsung di browser tanpa instalasi. Seluruh data tersinkronisasi secara real-time ke semua perangkat yang mengakses sistem secara bersamaan.

4. Firebase Authentication — Register & Login

✅ Register

  • Fungsi: createUserWithEmailAndPassword()
  • File: assets/js/login.js
  • Validasi email format & password minimal 6 karakter
  • Setelah register → email verifikasi dikirim otomatis
  • Akun langsung di-sign out hingga email diverifikasi

✅ Login

  • Fungsi: signInWithEmailAndPassword()
  • File: assets/js/login.js
  • Cek emailVerified — jika belum, login ditolak
  • Session disimpan di sessionStorage
  • Auth guard via onAuthStateChanged() di setiap halaman

Saat login, sistem memeriksa status emailVerified. Jika belum diverifikasi, akun otomatis di-sign out dan user diminta untuk memeriksa email terlebih dahulu.

6. Fitur Tambahan — Cloudflare R2 Storage ⭐

Sebagai layanan cloud tambahan di luar Firebase, proyek ini mengintegrasikan Cloudflare R2 sebagai object storage untuk menyimpan file foto. R2 dipilih karena tidak mengenakan biaya egress (bandwidth keluar) berbeda dengan Firebase Storage.

Konfigurasi R2

  • Worker URL: alp-r2-uploader.workers.dev
  • Public bucket via Cloudflare CDN
  • Bearer token authentication
  • File: assets/js/r2-service.js

Penggunaan

  • Foto inventaris alat lab
  • Foto bukti laporan kerusakan
  • Foto inventaris per ruangan
  • URL foto disimpan ke Firebase DB
// assets/js/r2-service.js — upload foto ke Cloudflare R2 async function uploadToR2(file, folder = "uploads") { const filename = `${folder}/${Date.now()}-${file.name}`; const response = await fetch(R2_WORKER_URL, { method: "PUT", headers: { "Authorization": `Bearer ${R2_AUTH_TOKEN}`, "X-File-Name": filename, "Content-Type": file.type, }, body: file, }); // URL publik disimpan ke Firebase Realtime Database return `${R2_PUBLIC_URL}/${filename}`; }
